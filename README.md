# Text
> :memo: A simple file manager
# What i need
##### :snake: Python 3x
##### :octocat: Git
# How to use
```py
from Text import Text, Readfile, text
import asyncio

class Aplicacao:
    text.config(on_ready="Ola Mundo!", filename="Ola.txt")
    Readfile.read_txt()

if __name__ == '__main__':
    asyncio.run(Text.run(Aplicacao))
```
